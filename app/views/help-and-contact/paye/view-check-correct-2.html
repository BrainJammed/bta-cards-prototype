Payroll software confirmations

You can also check or view submissions in HMRC's Basic PAYE Tools (BPT) software.

To check if HMRC received a submission, select 'View successful submissions' from your employer homepage.

To view a specific submission, follow these instructions in the user guide, but select 'View' instead of 'Change':

  • view an FPS (page 25)
  • view an EPS (page 37)

Contact your software provider if you do not use BPT.

Correct a PAYE submission

You must use your payroll software to correct Full Payment Submissions (FPSs) or Employer Payment Summaries (EPSs). HMRC staff cannot make changes for you.

Submissions for the current tax year

You can make changes to submissions for the current tax year until April 19.

Follow the user guide if you use HMRC's Basic PAYE Tools software (BPT):

  • correct an FPS (page 25)
  • correct an EPS (page 37)

Contact your software provider if you do not use BPT.

Submissions from an earlier tax year

If you spot an error in a submission after the tax year has ended, you need to submit an 'Earlier Year Update' if it is after April 19.

You can submit an Earlier Year Update using Basic PAYE Tools if you use BPT to manage your payroll.

If you use commercial software to manage your payroll, check with your software provider whether it can process Earlier Year Updates.

If your software doesn't offer Earlier Year Updates, you will need to install BPT to submit an Earlier Year Update.
